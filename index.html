Abir Chat Bot

Unofficial Messenger account bot is running.

Health

GET /healthz → ok

Commands

In Messenger, use the prefix / by default.

  • /help — list commands or details for a command
  • /admin — show Abir’s info
  • /ping — latency check
  • /echo <text> — echo message

Setup

  1. Put your Facebook appstate.json next to this file (path is configurable in config.json).
  2. Update config.json OWNER links/photo and ADMINBOT UID.
  3. Run npm start.
